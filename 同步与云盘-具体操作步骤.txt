================================================================================
  同步文件夹 + 云盘 具体操作步骤（数据互通）
================================================================================

一、电脑端：选择同步文件夹并上传到云盘
--------------------------------------

1. 准备同步盘文件夹（只需做一次）
   • 安装夸克网盘（或百度网盘/阿里云盘等），并开启「同步盘」。
   • 在同步盘里新建一个文件夹，例如：
     D:\QuarkSync\克丽斯腾
     或  C:\Users\你的用户名\QuarkSync\克丽斯腾
   • 该文件夹会和云端自动同步，你在里面放的文件会出现在云端和其他设备。

2. 在网站里「选择同步文件夹」
   • 用电脑浏览器（推荐 Chrome 或 Edge）打开本站（本地 index.html 或 Vercel 地址）。
   • 点击任意板块进入详情页（如「角色简介」「随笔记录」等）。
   • 点击顶部或工具栏的「一键导入/导出」。
   • 在弹窗里找到「同步盘（电脑端）」区域，点击「选择同步文件夹」。
   • 在系统弹窗中选中你在第 1 步建好的文件夹（如 D:\QuarkSync\克丽斯腾），点「选择文件夹」。
   • 页面会提示「已选择同步文件夹」。

3. 保存数据到同步文件夹（即上传到云盘）
   • 选好文件夹后，点击「保存到同步文件夹」。
   • 网站会在该文件夹里生成/覆盖「克丽斯腾-data.json」。
   • 夸克（或其它同步盘）会自动把这个文件同步到云端。
   • 以后在电脑上改完内容，再点一次「保存到同步文件夹」即可更新云端。

4. 从同步文件夹读取（从云盘拉取最新数据）
   • 在同一浏览器里，打开「一键导入/导出」→ 点击「从同步文件夹读取」。
   • 会读取该文件夹内的「克丽斯腾-data.json」并应用到当前页面。
   • 若在别的设备或电脑上改过并已同步，先等云盘同步完成，再点「从同步文件夹读取」即可拿到最新数据。

注意：关闭浏览器后再打开，需要重新点一次「选择同步文件夹」；选好后可反复「读取」和「保存」。


二、手机端：与云盘数据互通（无「选择同步文件夹」时）
----------------------------------------------------

1. 确保云盘里已有 克丽斯腾-data.json
   • 先在电脑上按上面步骤「保存到同步文件夹」一次，让夸克把文件同步到云端。
   • 在手机打开夸克 App，确认同步盘里能看到「克丽斯腾-data.json」。

2. 在手机浏览器或安装好的 App 里导入数据
   • 打开本站（或已安装的 PWA/APK）。
   • 进入任意板块详情 → 「一键导入/导出」。
   • 点击「从文件导入」→ 在文件选择器里找到夸克同步盘目录下的「克丽斯腾-data.json」→ 选择打开。
   • 数据会加载到当前页面。

3. 在手机修改后导出，让云盘同步
   • 改完内容后，在「一键导入/导出」里点击「导出 JSON」。
   • 手机会下载「克丽斯腾-data.json」。
   • 在下载管理或文件 App 里，把该文件「移动/复制」到夸克同步盘文件夹（或通过分享保存到夸克）。
   • 夸克会同步到云端；电脑上同步盘文件夹更新后，用「从同步文件夹读取」即可拿到手机上的修改。


三、数据互通流程小结
--------------------

• 电脑 → 云盘：选择同步文件夹 → 保存到同步文件夹（自动同步到云端）。
• 云盘 → 电脑：从同步文件夹读取。
• 手机 → 云盘：导出 JSON → 把文件存到夸克同步盘目录。
• 云盘 → 手机：在夸克里等同步完成 → 站点里「从文件导入」选 克丽斯腾-data.json。

• 多设备不要同时改同一份数据；建议「谁改谁先保存，另一台再读取」。


四、不支持「选择同步文件夹」时（部分浏览器）
--------------------------------------------

若浏览器没有「选择同步文件夹」按钮或点不了（例如部分国产浏览器、非 HTTPS 环境）：
• 电脑端也用「从文件导入」+「导出 JSON」：
  - 导出 JSON 后，手动把「克丽斯腾-data.json」复制到夸克同步盘文件夹。
  - 需要加载时，在一键导入/导出里选「从文件导入」，选同步盘里的 克丽斯腾-data.json 即可。
